# Outreach log

A simple, flat file for tracking the local businesses you've already reached out to.
**No database** — just open `contacts.csv` in Excel, Numbers, or Google Sheets.

## Columns

| Column | What it's for |
|---|---|
| **Business** | Business name |
| **Contact** | Person you spoke with |
| **Type** | dentist / lawyer / realtor / medical / etc. |
| **Email**, **Phone** | Best way to reach them |
| **Date contacted** | `YYYY-MM-DD` of first outreach |
| **Channel** | email / call / walk-in / referral |
| **Status** | Where they are in the pipeline (below) |
| **Notes** | Anything useful for the next touch |

## Status pipeline

```
contacted → replied → call booked → proposal sent → won / passed
```

## Tips

- Sort by **Status** to see who needs a follow-up.
- One row per business — update **Status** and **Notes** as things move.
- The two rows in the file are examples. Delete them and add your own.
